# brs-utils
Utilities for Roku development

This includes:

* A testing framework like jUnit
* A logger inspired in Log4j
* A i18n framework for translations
* An example channel with all the different templates and a video player

## buTypeUtils

* isString
* isBool
* isInt
* isArray
* isList
* isFloat
* isDouble
* isComparable

## buStringUtils

* contains
* indexOf
* lastIndexOf
* join
* toArray

* toString
